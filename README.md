# Terminated

A static status page shown when servers are unavailable.

![project status](https://img.shields.io/badge/project_status-completed-success?style=for-the-badge)
[![live site](https://img.shields.io/badge/live_site-blue?style=for-the-badge)](https://terminated.vercel.app/)

## Tech Stack

### Frontend

- **UI library**: [React 19](https://react.dev/)
- **Framework**: [Next.js 16](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)

### Backend

- **CDN**: [Vercel](https://vercel.com/docs/cdn)

## How to Update

To check for outdated packages, run `npm outdated`.

```bash
# Update dependencies (this will rewrite package-lock.json and package.json)
npm update --save
npm install --save-exact --save-dev prettier@latest prettier-plugin-tailwindcss@latest
```
